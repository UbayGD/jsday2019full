## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## DDBB
I'm using this for create the DDBB

```bash
$ docker run --name jsday -p 5432:5432 -e POSTGRES_USER=jsday -e POSTGRES_PASSWORD=jsday -e POSTGRES_DB=jsday -d postgres:11

```
